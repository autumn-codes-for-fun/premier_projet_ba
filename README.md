#premier_projet_Business-Analyste
##Description
Ce projet est une introduction à l'analyse de données avec Pandas

Pour cela j'ai téléchargé un jeu de données sur Kaggle que j'ai nettoyé dans le but de repondre à quelques problématiques business.

##Technologies utilisées
Pandas: nettoyage de données

##Objectif pédagogique
Ce projet m’a permis de :

Manipuler des données avec Pandas
Nettoyer un jeu de donnée

##Prochainement
Je vais appliquer une analyse exploratoire de données sur ce jeu de données afin de repondre à quelques porblématiques business
